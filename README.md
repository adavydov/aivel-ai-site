# Aivel — публичный сайт

Статический многостраничный сайт на Astro. Главная концепция: Aivel помогает раньше замечать значимые изменения, показывает проверяемое основание и позволяет задать следующий вопрос.

## Запуск

```powershell
npm install
npm run dev -- --host 127.0.0.1
```

Откройте `http://127.0.0.1:4321/`.

## Проверка

```powershell
npm run check
npm test
npm run build
npm run preview -- --host 127.0.0.1
```

Публичный адрес демонстрационной версии: `https://aivel-ai-site.pages.dev`.

Формы пока ничего не отправляют и не сохраняют. После отправки показывается только демонстрационное подтверждение в браузере. Перед полноценным привлечением клиентов необходимо подключить CRM и утвердить юридические документы.
